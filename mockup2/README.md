# Parafiles Static Theme Mockup 2

Open `index.html` in a browser. No build step, server, or network access is required.

The prototype is a static hash-routed demo with mock data and a pretend staff user:

- User: `admin.parafiles`
- Role: staff moderator and uploader
- Password shown in the mock sign-in screen: `demo-only`

Routes included:

- `#home`
- `#login`
- `#dashboard`
- `#quick-share`
- `#public-file`
- `#public-folder`
- `#account`
- `#moderation`
- `#users`
- `#operations`
- `#audit`
- `#style`

The visual direction is based on `../PARALIVES_THEME_NOTES.md`: cream surfaces, sage structure, coral actions, sky/lavender/peach accents, dark image-led hero treatment, and compact operational layouts.

This second version is intentionally tighter than `../mockup`: smaller gaps and padding, shorter controls, and rounded-rectangle buttons instead of full pill buttons.
